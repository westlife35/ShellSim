###功能介绍



>这是一个简易bash shell程序(运行./shellSim)，功能包括：

> * 执行shell内部的一些显示命令如ls，cal等。

> * 切换目录cd。

> * 输入输出重定向。

> * 执行管道命令。

> * 改变命令提示符，提示符最大支持长度为9。

> * 记录输入过的命令到历史记录。

> * 输入命令末尾加&符号，可以在后台运行。

> * tab补全命令的功能

> * jobs和bg功能

> * 此外，此程序支持更多的解析，可以在shell中的命令，同样都可以在这里解析并执行。
